# Evaluate Section - Quick Start Guide

## Overview
The Evaluate section helps you compare homes and conduct inspections. Three tabs: Browse, Compare, Inspection.

---

## Quick Actions

### Add a Home
1. Click "Browse" tab
2. Click "+ Add Home" button
3. Fill in address, price, bedrooms, bathrooms
4. Click "Add Home"

### Compare Homes
1. Select 2-3 homes (checkbox on cards)
2. **Swipe left** or click "Compare (X)" tab
3. View side-by-side comparison
4. Scroll to see all 76 criteria

### Inspect a Home
1. **Swipe left twice** or click "Inspection" tab
2. Select home from dropdown
3. Expand categories (Bathroom, Kitchen, etc.)
4. Rate each item: Good / Fix / Replace
5. Add notes as needed
6. View progress at top

## Key Features

### Browse Tab
- **Search**: Filter homes by address or neighborhood
- **Favorite**: Heart icon to mark favorites
- **Compare**: Select 2-3 homes (max 3)
- **Card Click**: View/edit home details

### Compare Tab
- **Columns**: One per home (max 3 visible)
- **Categories**: 10 categories, 76 items total
- **Visual Indicators**:
  - ‚úÖ Green = Good
  - ~ Yellow = Fair
  - ‚úó Red = Poor
  - ‚Äî Gray = Unrated
- **Pagination**: Arrow buttons if > 3 homes
- **Sticky Header**: Home details stay visible

### Inspection Tab
- **Categories**: 10 categories, 92 items total
- **Ratings**: Good (green), Fix (yellow), Replace (red)
- **Progress**: Real-time tracking at top
- **Filters**: All / Good / Fix / Replace / Not Rated
- **Notes**: Per-item notes (200 char) + section notes (500 char)
- **Expand/Collapse**: All or per-category
- **Back to Top**: Floating button after scrolling

## Common Issues

### "Can't add home"
- Check if address already exists
- Ensure all required fields filled
- Check internet connection
- Try again in a few seconds

### "Can't compare more than 3"
- Deselect one home first
- Limit is 3 homes at once
- Compare in batches if needed

### "Swipe not working"
- Ensure touch events enabled
- Try slower swipe
- Check if at end (Browse left, Inspection right)
- Try tapping tabs instead

### "Loading forever"
- Check internet connection
- Refresh page
- Clear browser cache
- Check Supabase status
